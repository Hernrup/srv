Configuration
=================
This configuration should be provided through env variables.

```
DUPLICITY_PASSPHRASE=
B2_ACCOUNT_ID=
B2_ACCESS_TOKEN=
B2_BUCKET=
MAILGUN_API_KEY=
MAILGUN_DOMAIN=
MAILGUN_RECIPIENT=
```

These environment variables should be set by parrent env
```
DATA_STORAGE_LOCATION
```
